# dst_fc_mod
DST FileCommands mod

Мод читает команды из файла и воспроизводит в игре (с частотой примерно 3 раза в сек).
Поддерживаются команды:
- дать игроку предмет givePlayer
- заспаунить моба\предмет
  * spawnNearPlayer - в радиусе 6 от игрока (fab, randomBoss, randomBossLight, randomBossHard)
  * spawnAtPlayer - на текущей позиции игрока (используется для запуска метеоритов в игрока)
- выполнить команду для игрока playerCommand (speed, damage, freeze, charge)
- вызвать событие playerEvent/worldEvent
- телепорт в рандомное место teleportRandom

Все команды, которые имеют отношение к игроку, выполняются для всех игроков на сервере.
Команды можно писать в одну строку через ";".
Помимо вышеперечисленных команд, можно выполнять любые, которые выполняются в консоли, 
однако они будут выполнены однократно, а не для всех игроков.

Настройки мода:
- Temp\cmd.txt - читает из файла C:\Temp\cmd.txt (не работает!!!)
- DST\cmd.txt - читает из файла в папке с игрой DST\cmd.txt (для сервера путь scripts\..?)
- Custom - относительный(!) путь вводится при активации мода в игре
